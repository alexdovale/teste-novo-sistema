Política de Privacidade

Sistema de Gerenciamento de Pauta (SIGAP)

1. Introdução

O Sistema de Gerenciamento de Pauta (SIGAP) foi desenvolvido para auxiliar na organização e gestão de atendimentos jurídicos. Esta política descreve como os dados dos assistidos e colaboradores são tratados.

2. Coleta de Dados

Coletamos apenas as informações estritamente necessárias para a realização do atendimento jurídico e controle de fluxo, tais como:

  • Nome completo e identificação dos assistidos;
  • Dados de contato (telefone, endereço);
  • Informações sobre a demanda jurídica;
  • Dados de cadastro dos colaboradores (nome, e-mail).

3. Uso das Informações

As informações são utilizadas exclusivamente para:

  • Gerenciamento da fila de atendimento (Kanban);
  • Geração de relatórios estatísticos anonimizados;
  • Emissão de documentos e checklists necessários para a assistência jurídica;
  • Controle de acesso e segurança do sistema.

4. Armazenamento e Segurança

Os dados são armazenados em nuvem segura (Google Firebase), com regras de acesso restritas. Apenas usuários autenticados e autorizados pela administração podem acessar dados sensíveis. O sistema utiliza criptografia padrão na transmissão de dados (HTTPS).

5. Compartilhamento

Não compartilhamos dados pessoais com terceiros para fins comerciais. O compartilhamento de informações ocorre apenas no âmbito da prestação de assistência jurídica ou por determinação legal.
